# 🎵 Bollywood Music Recommendation System

A content-based music recommendation system that suggests similar Bollywood songs based on your input.

## 🔗 Live Demo
[Click here to try the app](https://abhinavsinha00-music-recommendation-system--app-gecvzs.streamlit.app/)

## 📌 About
This project recommends similar Bollywood songs using Machine Learning.
By the end of this project, you'll understand how platforms like **Spotify, Gaana, and Apple Music** use ML to recommend music!

## 🔹 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 📚 Topics Covered
- Data Cleaning & Preprocessing
- Feature Selection & Encoding
- Similarity-Based Recommendation (Cosine Similarity)
- TF-IDF Vectorization
- Generating Personalized Song Recommendations

## 🚀 How It Works
1. Enter a Bollywood song name
2. The model finds similar songs based on artist, album, and genre
3. Top 5 recommendations are displayed with thumbnails

## 📁 Dataset
Bollywood Songs Dataset with Album & Genre information
